This calculator uses the barometric formula:

Altitude = ( (P₀/P)1/5.257 - 1 ) × (T + 273.15) / 0.0065

Where:
P₀ = Sea level pressure (hPa)
P = Current pressure (hPa)
T = Temperature (°C)
